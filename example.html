jquery.placeholder

A jQuery plugin to simulate HTML5 placeholder attribute on input elements.

Just create an input with a placeholder attribute

<input type="text" placeholder="This is a placeholder text" id="placeholder_me" />

And then do some javascript awesomess

<!-- Remember to include jQuery too! -->
<script src="jquery.placeholder.js" type="text/javascript" charset="utf-8"></script>
<script type="text/javascript" charset="utf-8">
  $('#placeholder_me').placeholder();
</script>

Live demo

Try clicking inside the input and watch how the placeholder text just dissapears
